LeetCode 332周赛

6354. 找出数组的串联值

解答：双指针

6355. 统计公平数对的数目

解答：排序+双指针

6356. 子字符串异或查询

解答：哈希表+遍历，这道题是一道脑筋急转弯的题目，看着搜索空间很大，其实有一个重要的提示：

  • 0 <= firsti, secondi <= 10^9

所以每个s的位置记录向前的31位数字就可以了。
